# SolidX - Intelligent Scaffolding for Enterprise Applications

**Enterprise software should ship fast - and stay clean.**

SolidX delivers the speed of low-code with the flexibility of custom development - producing standards-based, extensible **NestJS + React** systems your engineers can own. No compromises, no vendor lock-in, no black box.

---

## What is SolidX?

SolidX is an **open-source, metadata-driven scaffolding platform** that turns your data models into production-grade enterprise applications - with auth, APIs, roles, and admin views - in minutes, not months.

Define your modules, models, and fields through the admin portal or describe them to the AI Agent. Run `solidctl` to generate a complete, clean codebase that your team can read, test, and extend.

### What it is

- **A metadata-first scaffolding engine** that generates production-grade NestJS + React code from your data models
- **Batteries included** - built-in IAM, notifications, queuing, media management, and auto-generated REST APIs
- **An AI Agent** (GenAI-powered, MCP backend) that generates metadata updates or custom code, immediately applicable in the admin portal
- **Fully open-source output** you can fork, deploy, extend, and own - no runtime dependency on SolidX

### What it isn't

- Not a closed low-code "black box" that hides your logic behind proprietary abstractions
- Not "prompt-to-production" code roulette - every generation follows structured metadata and clean architecture
- Not a new proprietary stack you can't hire for - it's NestJS, React, TypeORM, and PostgreSQL

## The Platform

| Capability | What you get |
|---|---|
| **Admin Portal & App Builder** | Define modules, models, and fields - configure your entire data layer without writing SQL |
| **20+ Field Types** | Short text, rich text, dates, booleans, media, computed fields, and full relational support |
| **Automated Schema Generation** | Metadata definitions translate directly into PostgreSQL tables, constraints, and foreign keys via `solidctl` |
| **Instant REST APIs** | Every model gets full CRUD endpoints with filtering, sorting, pagination, and auth guards |
| **Auth - Out of the Box** | Password, OTP, and OAuth (Google) with login, registration, password reset, and session management |
| **Roles & Access Control** | Role-based permissions with model-level filters, record rules, and field-level access controls |
| **AI Agent** | GenAI-powered chat sidebar that generates metadata or custom code - apply changes instantly |
| **Model Configuration** | Soft delete, audit logs, i18n, naming conventions, export controls - per model, zero SQL |
